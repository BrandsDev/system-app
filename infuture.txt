<?php
function generateNumbers($start, $end) {
    for ($i = $start; $i <= $end; $i++) {
        $formattedNumber = '+880171' . str_pad($i, 7, '0', STR_PAD_LEFT);
        echo $formattedNumber . PHP_EOL;
    }
}

$startNumber = 2001;
$endNumber = 3000;
generateNumbers($startNumber, $endNumber);
?>

SSL Configure 
---------------
To remove all Let's Encrypt SSL certificates related to the ilannoor.institute domain and generate a new one, you can follow these steps:

    Stop the web server:

    arduino

sudo systemctl stop apache2

Remove the existing SSL certificate files:

bash

sudo rm -rf /etc/letsencrypt/live/ilannoor.institute
sudo rm -rf /etc/letsencrypt/archive/ilannoor.institute
sudo rm -rf /etc/letsencrypt/renewal/ilannoor.institute.conf

Remove the Certbot cron job for certificate renewal:

sudo crontab -e

Locate the line that refers to the Certbot renewal for the ilannoor.institute domain and remove it. Save and exit the editor.

Generate a new SSL certificate using Certbot:

css

sudo certbot certonly --apache -d ilannoor.institute -d www.ilannoor.institute

This command will request a new SSL certificate for the ilannoor.institute domain and its www subdomain, using the Apache authenticator.

Follow the prompts to complete the certificate generation process. Make sure to provide valid email address and agree to the terms of service.

Restart the web server:

sql

sudo systemctl start apache2

Configure Apache2: 
sudo a2ensite yourdomain.com-le-ssl.conf
sudo service apache2 reload
